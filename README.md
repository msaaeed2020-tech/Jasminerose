# 🌸 متجر وردة الياسمين - الوثائق التقنية

هذا المستودع مخصص لإدارة الأصول البرمجية، إعدادات السيو (SEO)، والربط التقني لمتجر **وردة الياسمين** على منصة **توسعة (Twsaa)**.

## 🛠 الحالة التقنية الحالية
- **المنصة:** توسعة (Twsaa).
- **حالة الأرشفة:** تم إعداد ملفات `sitemap.xml` و `robots.txt`.
- **أدوات التحليل:** مرتبط بـ Google Search Console و Google Analytics.
- **التسويق التقني:** العمل جارٍ على ربط TikTok Marketing API.

## 📂 هيكلة الملفات التقنية
- `/seo`: يحتوي على نسخ احتياطية من إعدادات محركات البحث.
- `/marketing`: وثائق معرفات الإعلانات (Advertiser IDs) وإعدادات البكسل.
- `/assets`: أي ملفات CSS مخصصة لتحسين مظهر المتجر.
